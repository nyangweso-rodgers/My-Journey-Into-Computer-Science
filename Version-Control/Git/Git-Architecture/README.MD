# Git Architecture

# Check Git Components
* Inside the project which contains a Git repository, we can check the Git components:
    ```git
        ls -F1 .git/
    ```
* __Output__:
  * COMMIT_EDITMSG
  * config
  * description
  * HEAD
  * hooks/
  * index
  * info/
  * logs/
  * objects/
  * packed-refs
  * refs/
